====================================================
DataSphereXX  |  Project Overview
====================================================

This repository hosts the DataSphereXX web hub and
Netlify serverless functions.

STRUCTURE
----------
index.html              – main landing page
_netlify.toml           – Netlify build configuration
netlify/functions/      – serverless backend scripts
    getLinks.js
    updateLinks.js
    deleteLink.js
    restoreLast.js

DEPLOYMENT
-----------
Hosted via Netlify.
Functions: netlify/functions
Publish: root directory
